# HZTIKU
该工具用于对于特定格式题库word文件进行xlsx文件格式转换
## 安装

要使用此工具，您需要在系统上安装 Python 3 以及以下软件包：

- docx
- openpyxl
- re

您可以通过运行以下命令来安装这些软件包
```
pip install -r requirements.txt
```

git clone本git

```
git clone https://github.com/grandjohnson/HZTIKU.git
```
升级到新版
```
cd HZTIKU
git pull
pip install -r requirements.txt
```
